# 🚀 Pro VSCode Snippets

A collection of useful and productivity-focused Visual Studio Code snippets for modern web development.

## ✨ Features

- ⚡ Fast and reusable code snippets
- 📦 Multiple language support
- 🧩 Easy to install
- 🚀 Improves development speed
- 🔥 Frequently updated

## 📚 Supported Languages

- JavaScript
- HTML
- React
- Global (available in any language)

> More languages and frameworks will be added over time.

## 📦 Installation

### Option 1: Clone the repository

```bash
git clone https://github.com/alvarosiles/pro-vscode-snippets.git
```
